# 購物車系統

本專案為一個模擬購物車系統，作為 Practice 1 作業的一部分。該系統模擬了購物過程，實作了商品列表、加入與刪除商品、查看購物車、計算總金額等功能，並進一步擴展至結帳、使用折扣碼與幣值轉換。

## 功能

### 階段 1 - 基本購物車系統
1. **商品列表** - 顯示可供購買的商品名稱與價格。
   - 商品範例如下：
     - 潛水相機防丟繩 - TWD 199
     - 潛水配重帶 - TWD 460
     - 潛水作業指北針 - TWD 1100

2. **新增至購物車** - 允許使用者根據商品及數量新增至購物車。
   - 商品代號限制：1-3
   - 數量限制：1-5

3. **自購物車刪除** - 允許使用者選擇欲刪除的商品及數量。
   - 商品代號限制：1-4

4. **查看購物車** - 顯示當前購物車內容。
5. **計算總金額** - 計算購物車內商品的總金額。
6. **退出網站** - 結束程式。

### 階段 2 - 擴展購物車系統
1. **結帳** - 使用者可以選擇結帳，確認訂單細節，並輸入折扣碼。
   - 支援的折扣碼：1111（享有 5% 折扣）
   
2. **轉換幣值** - 可將總金額轉換為不同幣值（USD、CNY、JPY），使用固定匯率：
   - TWD 轉 USD: 0.031
   - TWD 轉 CNY: 0.23
   - TWD 轉 JPY: 4.59

## 錯誤處理
系統包含全面的錯誤處理，包括以下情況：
- 無效的商品或數量輸入
- 庫存不足時的提醒
- 無效的付款選項或折扣碼輸入
